# Proyecto de Estructura de Datos II - 2026

---

## Descripción General

Este repositorio contiene el código y la documentación del proyecto de estructura de datos 2 desarrollado entre los meses de Mayo y Junio de 2026. El objetivo del proyecto es implementar el conocimiento adquirido en el transcuro de la catedra de Estructura de Datos II de la Universidad de El Salvador.

---

### Descripción del Proyecto

El proyecto consiste en un sistema gestor de cobro de agua, el cual permite lo siguiente:

- Registrar Empleados: El sistema permite ingresar nuevos empleados a los cuales se les genera credenciales automaticamente para acceder al sistema
- Registrar nuevos clientes: Permite registrar nuevos clientes
- Asignar contratos: Se permite asignar nuevos contratos a los clientes junto a su medidor
- Generar Factura: El sistema puede generar una factura en base a una lectuta previa
- Pagar Factura: El sistema permite pagar las facturas generadas
- Historial de pago: Permite al usuario ver su historial de pago

Toda la informacion esta siendo almacenda en una base de datos

---

### Tecnologías Utilizadas

- Lenguaje de Programación: Java.
- Gestor de Bases de Datos: Postgres
